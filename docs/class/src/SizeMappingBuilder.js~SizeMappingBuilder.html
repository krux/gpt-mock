    public build(): SizeMappingArray source

    Builds a size map specification from the mappings added to this builder. If any invalid mappings have been supplied, this method will return null. Otherwise it returns a specification in the correct format to pass to {@link Slot#defineSizeMapping()}. The behavior of the builder after calling SizeMappingBuilder#build is undefined.

    Return:

    SizeMappingArray (nullable: true)

    The result built by this builder. Can be null if invalid size mappings were supplied.
